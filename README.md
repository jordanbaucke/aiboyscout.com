# 21st Century Digital Boy Scout

A simple, minimalistic website exploring technology, ethics, and digital citizenship.

## About

This site is designed to be intentionally minimalistic, focusing on simplicity, performance, and foundational web technologies.

## Technologies

* HTML5
* CSS3
* JavaScript (vanilla)
* GitHub Pages for hosting

## Local Development

To run the site locally with auto-refresh on file changes:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

The site will open in your browser at `http://localhost:3000` and automatically refresh when you make changes to any files.

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.
